## 7. 综合应用链条
### 7.1 监督学习链条
- 函数表示 → 损失函数 → 梯度 → 梯度下降 → 参数更新

### 7.2 深度网络链条
- 矩阵乘法 → 仿射变换 → 激活函数 → 非线性复合 → 深度神经网络

### 7.3 卷积视觉链条
- 二维卷积 → 卷积核 → 局部特征 → 层级语义 → CNN视觉任务

### 7.4 正则化链条
- 范数 → 正则化项 → 模型复杂度控制 → 泛化能力提升

### 7.5 稀疏学习链条
- L0伪范数 → L1松弛 → 稀疏优化 → 特征选择

### 7.6 概率学习链条
- 概率分布 → 熵 → 交叉熵/KL散度 → 分类模型/生成模型训练

### 7.7 相似性检索链条
- 内积/范数 → 距离/相似度 → 聚类/排序/推荐/检索
